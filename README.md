# chromatron.io
Chromatron - Wireless Pixel Control


## Documentation

### Preview locally
```sh
$ gulp serve
```

### Build the site
```sh
$ gulp build
```

### Deploy to GitHub Pages
```sh
$ gulp deploy
```

For more information on the deploy task, see documentation for the NPM Package `gulp-gh-pages` at https://github.com/shinnn/gulp-gh-pages.
